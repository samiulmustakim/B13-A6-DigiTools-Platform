## 📌 Project Name
**B13-A6-DigiTools-Platform**

---

## 📝 Description
DigiTools Platform is a web app where users can explore digital tools and easily purchase them through an interactive card-based interface.

---

## ⚙️ Technologies Used
- React.js  
- Vite  
- Tailwind CSS  
- DaisyUI  
- React Toastify  
- React Icons  

---

## ✨ Features

### 🛒 Add to Cart Functionality
Users can select tools and add them to a cart, making it easy to manage chosen items.

### ⚡ Fast & Responsive UI
Built with Vite and Tailwind CSS, the website is very fast and works smoothly on all devices.

### 🔔 Toast Notification System
User actions like adding items show instant feedback using React Toastify.
